# 📚 Library Project - Frontend

Bienvenido al repositorio oficial de la interfaz de usuario para **Library**. Este proyecto se enfoca en el desarrollo visual y la arquitectura de navegación para un sistema de gestión bibliotecaria.

Este repositorio contiene la estructura de componentes y el diseño responsivo necesario para interactuar con el sistema.

---

## 🛠️ Stack Tecnológico
Para la construcción de la interfaz, hemos utilizado:

* **Framework:** [React.js](https://reactjs.org/) (v18+) con **Vite** para una carga optimizada.
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/) para el desarrollo de componentes visuales modernos y personalizados.
* **Navegación:** [React Router Dom](https://reactrouter.com/) para la gestión de las rutas y el flujo entre páginas.

---

## 🚀 Alcance del Desarrollo (Frontend)
Durante esta etapa, el equipo se centró en la creación de la arquitectura visual y funcional del cliente:

1.  **Interfaz de Autores y Libros:** Diseño y maquetación de los contenedores, tablas y formularios necesarios para la visualización de la información.
2.  **Arquitectura de Rutas:** Configuración de la navegación dinámica para permitir el desplazamiento entre los diferentes módulos del sistema.
3.  **Componentes UI:** Creación de elementos reutilizables siguiendo los lineamientos definidos en el prototipado.
4.  **Maquetación Responsiva:** Implementación de estilos con Tailwind para asegurar que la "cajita" de la aplicación se adapte a diferentes tamaños de pantalla.

## 💻 Instalación y Configuración Local

Siga estos pasos para desplegar el entorno de desarrollo:

1.  **Clonación del repositorio:**
    ```bash
    git clone [https://github.com/ResilenteMG/mi-biblioteca-front.git](https://github.com/ResilenteMG/mi-biblioteca-front.git)
    ```

2.  **Instalación de dependencias:**
    ```bash
    npm install
    ```

3.  **Ejecución del servidor local:**
    ```bash
    npm run dev
    ```
