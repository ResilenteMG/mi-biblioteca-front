# 📚 Library Project - Frontend

Bienvenido al repositorio oficial del cliente para **Library**, un ecosistema digital diseñado para la gestión eficiente de catálogos bibliográficos, autores y préstamos.

Este proyecto ha sido construido enfocándose en la modularidad, el rendimiento y una interfaz de usuario intuitiva.

---

## 🛠️ Stack Tecnológico
Para garantizar un desarrollo ágil y escalable, el proyecto utiliza las siguientes tecnologías:

* **Core:** [React.js](https://reactjs.org/) (v18+) con **Vite** como empaquetador de alto rendimiento.
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/) para una arquitectura de diseño basada en utilidades y diseño responsivo.
* **Navegamiento:** [React Router Dom](https://reactrouter.com/) para la gestión de rutas dinámicas (Single Page Application).
* **Comunicación:** [Axios](https://axios-http.com/) para el consumo eficiente de la API REST del backend.

---

## 🚀 Funcionalidades Principales
El sistema actualmente permite la gestión integral de los siguientes módulos:

1.  **Gestión de Autores (CRUD):** Registro completo, edición de perfiles y visualización de la base de datos de autores.
2.  **Control de Libros (CRUD):** Administración del catálogo, incluyendo títulos, disponibilidad y vinculación con autores.
3.  **Sistema de Rutas:** Estructura de navegación optimizada que separa las vistas administrativas de las de consulta.

## 📐 Planificación y Diseño
El éxito de la implementación se basa en una fase previa de estructuración técnica:
* **Prototipado UI/UX:** Diseños de alta fidelidad realizados en **Figma**.
* **Arquitectura:** Modelado de datos y flujo de información diseñado en **Draw.io**.
* **Metodología:** Gestión de Épicas, Historias de Usuario y Sprints a través de **Jira**.
* **Validación de API:** Consumo de endpoints previamente verificados con **Postman**.

---

## 💻 Instalación y Configuración Local

Siga estos pasos para desplegar el entorno de desarrollo:

1.  **Clonación del repositorio:**
    ```bash
    git clone [https://github.com/ResilenteMG/mi-biblioteca-front.git](https://github.com/ResilenteMG/mi-biblioteca-front.git)
    ```

2.  **Instalación de dependencias:**
    ```bash
    npm install
    ```

3.  **Ejecución del servidor local:**
    ```bash
    npm run dev
    ```
